Mis trabajos

Implementando CSS

En nuestro primer proyecto, nos enfocamos en aprender y aplicar CSS dentro de una página web. Exploramos los conceptos básicos, como la sintaxis y los selectores, y aprendimos a estilizar elementos HTML para mejorar su apariencia y usabilidad. Este proyecto nos brindó una sólida comprensión de CSS, preparándonos para futuros desarrollos web.

Implementación de Flex

En nuestro segundo proyecto, profundizamos en CSS al aprender a implementar Flexbox. Esta técnica nos permitió crear diseños web flexibles mediante propiedades como display: flex, justify-content y align-items. Aprendimos a organizar y alinear elementos de manera eficiente, mejorando así la estructura y la disposición de nuestros proyectos web. Este proyecto nos brindó una comprensión más avanzada de CSS y nos equipó con herramientas poderosas para crear diseños modernos y adaptables.

Proyecto Cuppon

En este proyecto, implementamos CSS y Flexbox para crear una página web responsiva. Utilizamos estas tecnologías para estilizar elementos y organizarlos de manera flexible en diferentes dispositivos y tamaños de pantalla. El resultado fue una página que se adapta bien a distintos dispositivos, mejorando la experiencia del usuario.
